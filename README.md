## Running the application

To run the application, follow these steps:

1. **Install dependencies**:
    ```bash
    npm install
    ```

2. **Start the application**:
    ```bash
    npm run dev
    ```

3. **Run tests** (optional):
    ```bash
    npm run test
    ```
